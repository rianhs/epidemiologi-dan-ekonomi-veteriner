## 💡 Apa isi situs web ini?
Situs web ini merupakan sumber pembelajaran untuk mata kuliah Epidemiologi dan Ekonomi Veteriner di Program Studi Kedokteran Hewan, Fakultas Kedokteran, Universitas Hasanuddin. 
* Github repository: <https://github.com/rianhs/epidemiologi-dan-ekonomi-veteriner>
* Github pages: <https://rianhs.github.io/epidemiologi-dan-ekonomi-veteriner/>

## 📁 Struktur Pembelajaran 
Pembelajaran di situs ini menggunakan pendekatan outcome-based education. Mahasiswa disarankan untuk menggunakan materi di situs ini secara berurutan berdasarkan urutan capaian pembelajaran di bawah ini. 
* Mata kuliah ini memiliki beberapa capaian pembelajaran mata kuliah (CPMK) dan subcapaian pembelajaran mata kuliah (Sub-CPMK).
* Materi disusun berdasarkan Sub-CPMK. Setiap Sub-CPMK memiliki satu modul dan dapat dilengkapi dengan beberapa simulasi interaktif, prosedur, tugas, dan dataset untuk praktik.

### CPMK-1
Mahasiswa mampu mengevaluasi status kesehatan populasi hewan dengan mengintegrasikan konsep epidemiologi deskriptif dan analitis.

#### Sub-CPMK-1
* Mahasiswa mampu menguraikan konsep dasar epidemiologi veteriner yang mencakup peran, ruang lingkup, pola kejadian penyakit, jenis pendekatan epidemiologi, dan prinsip estimasi demografi populasi hewan.
* Pekan: 1
* [Modul 01](modul/modul-01.md) 

#### Sub-CPMK-2
* Mahasiswa mampu menganalisis dinamika penyebaran penyakit pada populasi hewan berdasarkan konsep segitiga epidemiologi, model kompartemen, kurva epidemi, dan parameter penularan penyakit.
* Pekan: 2
* [Modul 02](modul/modul-02.md)

### CPMK-2
Mahasiswa mampu merancang program pengendalian dan penanggulangan penyakit hewan yang berbasis bukti dan layak secara ekonomi pada populasi hewan produksi, hewan kesayangan, dan/atau satwa liar.

#### Sub-CPMK-6
Mahasiswa mampu menilai kualitas program surveilans dan peringatan dini penyakit pada populasi hewan berdasarkan tujuan program dan karakteristik epidemiologis penyakit.

#### Sub-CPMK-7
Mahasiswa mampu menilai kualitas prosedur investigasi wabah penyakit hewan berdasarkan ketepatan definisi kasus, urutan langkah epidemiologis, hingga relevansi tindakan darurat.

#### Sub-CPMK-8
Mahasiswa mampu menilai kualitas strategi manajemen risiko terkait agen penyakit berdasarkan hasil analisis risiko impor pada lalu lintas hewan atau produk hewan dan penilaian risiko gabungan antarinstitusi.

#### Sub-CPMK-9
Mahasiswa mampu menilai dampak ekonomi penyakit dan kelayakan finansial program intervensi kesehatan hewan, baik pada tingkat kawanan hewan maupun tingkat regional atau nasional.

#### Sub-CPMK-10
Mahasiswa mampu menyusun dokumen strategi pengendalian dan penanggulangan penyakit hewan yang komprehensif berdasarkan sintesis data epidemiologis dan pertimbangan ekonomi.
